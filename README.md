# 💬 MERN Real-Time Chat Application

A full-stack real-time chat application built using MERN Stack, Socket.io and Google OAuth.

## 🚀 Features

- Google OAuth Login
- JWT Authentication
- Real-time messaging
- Private chat
- Group chat
- Online users
- User profiles
- Responsive UI

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Redux Toolkit
- React Router

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.io

### Authentication
- JWT
- Google OAuth 2.0

## 📸 Project

Real-time communication app with secure authentication and instant messaging.
